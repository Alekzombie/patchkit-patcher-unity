>> Thanks!

Thank you for downloading PatchKit Patcher!

PatchKit Patcher is a client of PatchKit service. PatchKit is a service to host and distribute game content to the players without any effort. Patcher is an application that will provide the access to the game, download it and make sure to update it whenever a new version is released.


--
>> Getting Started
--

If this is the first time you're using PatchKit Patcher, please visit:

http://docs.patchkit.net/unity_custom_patcher.html

Or click on the following menu item:

Tools / PatchKit Patcher / Getting Started...


Sorry, no offline docs. PatchKit is a service so you need to stay online to use it.
